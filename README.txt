# SpendsterX — Smart Expense Tracker

SpendsterX is a modern Flutter-based expense tracking app designed to help users manage their finances efficiently with powerful insights and a clean UI.

---

## Features

### Dashboard

* Today's spending overview
* Monthly progress tracking
* Daily safe spending calculation
* Smart status indicator (On Track / Careful / Overspent)

### Transactions

* Add income & expenses
* Edit and delete transactions
* Filter by date range
* Monthly default view

### Payment Methods

* Fully customizable (Bank, UPI, Cash, etc.)
* Dynamic system (user-defined methods)

### Balance Management

* Separate balance tracking for each method
* Real-time updates

### Insights

* Category-wise spending (Pie Chart)
* Weekly spending trends

### Backup System

* Manual export & import of data
* JSON-based storage
* Works offline

---

## Tech Stack

* Flutter (UI)
* Hive (Local Storage)
* fl_chart (Data Visualization)
